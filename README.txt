# Email Signature Generator

This is a simple web application that allows users to generate customized email signatures. Users can input their personal information and see a live preview of their email signature.

## Features

- Input fields for name, pronouns, job title, email, phone, and website
- Profile picture upload functionality
- Live preview of the email signature
- Responsive design

## Getting Started

To use this Email Signature Generator, follow these steps:

1. Clone this repository to your local machine:
   ```
   git clone https://github.com/your-username/email-signature-generator.git
   ```
2. Open the `index.html` file in your web browser.

Alternatively, you can host this application on any static web hosting service.

## Usage

1. Fill in your personal information in the form fields.
2. Upload a profile picture (optional).
3. The preview will update in real-time as you type.
4. Once you're satisfied with your signature, you can copy the HTML from the preview section to use in your email client.

## Customization

You can customize the appearance of the email signature by modifying the `styles.css` file. Feel free to adjust colors, fonts, and layout to match your preferences or brand guidelines.
